===============================================================
X4: Foundations - Spherical Long Range Scan Mod - By Sladernimo
===============================================================



------------
Information:
------------
This is a mod for the game "X4: Foundations" by "Egosoft". "X4: Foundations" was released in 2018.



------------
Description:
------------
If you want easier to use long range scanning mode where you don't have to rotate your ship in multiple directions to scan the area this mod is what you are looking for.
With this mod the long range scan works in all directions.



-------------
Installation:
-------------
If not already done create folder "extensions" inside your X4 Foundations documents directory (\Documents\Egosoft\X4\1638\extensions).
Copy the mod folder into the "extensions" folder.
Start the game and have fun :)



------
Usage:
------
After you installed the mod it is enabled by default.
You can enable/disable the mod in the main menu (in-game).
